# Third Gulf War OSINT Dataset

This repository contains a structured open-source intelligence dataset and analytical report tracking the Iran–Israel–United States conflict from October 2023 to March 2026.

## Contents

Dataset
• 20+ analytical sheets  
• 500+ structured conflict records  
• casualty tracking  
• weapons usage data  
• economic and supply chain impact  

Report
• conflict timeline and escalation analysis
• casualty assessment
• military capability analysis
• geopolitical alignment
• economic and oil market impact
• scenario modelling

## Methodology

Data compiled from open-source intelligence including:

• Reuters  
• AP  
• Bloomberg  
• UN OCHA  
• IAEA  
• Human Rights monitoring organizations  
• OSINT research groups  

All records include confidence levels and verification tiers.

## Disclaimer

This project uses open-source intelligence (OSINT).  
Some Phase 3 data (Feb–Mar 2026) remains preliminary and subject to revision.
